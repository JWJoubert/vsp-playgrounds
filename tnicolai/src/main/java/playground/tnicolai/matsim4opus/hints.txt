MATSim4UrbanSim hints:

- PyXB is required to use MATSim4UrbanSim
- Even when not using/updating the travel_data table (e.g. Brussels case study) it needs to be available 
  as data set for the get_matsim_data_into_chace model. The reason is that his model is inherited form a 
  travel model stub, for integrating arbitrary external travel models, that expects the travel_data data set.
